Module maildir

Source
Expand description

Unread mail. Only supports maildir format.

Note that you need to enable maildir feature to use this block:

cargo build --release --features maildir

§Examples

block = "maildir"
interval = 60
inboxes = ["~/mail/local", "~/maildir/account1/*"]

threshold_critical = 10
display_type = "new"
